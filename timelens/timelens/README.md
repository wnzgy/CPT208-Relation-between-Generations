# TimeLens | 时光透镜

> 🎓 CPT208 课程项目 - B4: 代际关系 (Relation between Generations)

**TimeLens** 是一款连接祖辈与孙辈的数字化记忆传承应用。通过扫描家中的老物件（怀表、老照片等），解锁封存其中的家族故事，实现跨越时空的代际对话。

![项目截图](./public/hero.png)

---

## ✨ 核心功能

### 📱 扫描解锁
- 模拟 NFC/AR 扫描实体物品
- 解锁家族珍贵记忆
- 3D 可视化呈现

### 💬 代际对话
- 孙辈与祖辈双向留言
- 不同身份视觉区分（👧 蓝色 / 👴 黄色）
- 实时对话时间戳

### 🗺️ 记忆地图
- 查看所有家族记忆
- 解锁进度追踪
- 待探索物品提示

### ➕ 创建记忆
- 上传新的家族故事
- 关联实体物品
- 跨代传承

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 pnpm

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```
打开浏览器访问 http://localhost:5173

### 构建生产版本
```bash
npm run build
```
构建结果在 `dist/` 文件夹

## 🎮 使用指南

### 1. 首页 - 扫描
- 点击中间的扫描按钮
- 模拟扫描过程（1.8秒）
- 自动解锁一个记忆

### 2. 3D 记忆展示
- 旋转查看 3D 记忆核心
- 点击「查看留言」展开对话面板
- 右上角切换身份（孙辈/祖辈）
- 输入留言并发送

### 3. 记忆地图
- 底部导航切换到「Map」
- 查看所有记忆（已解锁/待解锁）
- 点击已解锁的记忆可重新查看

### 4. 添加记忆
- 底部导航切换到「Add」
- 填写物品名称、年份、故事
- 保存后可在地图中查看

## 💾 数据存储

本项目使用 **LocalStorage** 进行本地数据持久化：
- 所有记忆数据保存在浏览器本地
- 刷新页面数据不丢失
- 支持多设备需配合后端（当前版本暂未接入）
