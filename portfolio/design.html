Design Process

From concept exploration to interaction decisions

Design Principles

1. Tangible First

Physical keepsakes and NFC taps should trigger the flow before any menu navigation.

2. Playful Discovery

Memories should feel unlocked and explored, not filed in a traditional archive.

3. Reciprocal Storytelling

Both generations need to speak, respond, and continue the story over time.

4. Accessible by Default

Interaction must remain understandable for users with different digital confidence levels.

Design Alternatives

Three directions compared before final selection

Alternative Comparison

Option Description Pros Cons Decision
Option A
Digital Archive		 Family tree and text-first memory management. Easy to build, structured records. Low engagement for children, weak interactivity. Not selected
Option B
Video Storybook		 Record and replay long-form video stories. Rich emotion and authentic voice. High production effort and editing overhead. Not selected
Option C
Tangible NFC + 3D		 Tap object to unlock memory, then continue with message exchange. Playful, cross-generational, anchored in real objects. Needs NFC support and careful mobile optimization. Selected
