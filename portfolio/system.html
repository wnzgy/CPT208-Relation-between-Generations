Engineering Notes

Implementation quality and constraints

Performance

  • 3D canvas tuned for mobile rendering budget
  • Smooth transition states to avoid abrupt context switch
  • Local-first data to keep interaction responsive

Compatibility

  • Web NFC path for supported Android browsers
  • Manual scan fallback for unsupported environments
  • Touch targets sized for classroom demo reliability

Risk and Mitigation

Risk: NFC capability differs by device model.
Mitigation: Fallback unlock path included in prototype.

Risk: 3D rendering overhead on low-end devices.
Mitigation: Controlled model complexity and simpler effect stack.
