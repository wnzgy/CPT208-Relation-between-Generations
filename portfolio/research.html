User Research and Evidence

Stakeholders, questionnaire results, and literature-grounded design decisions

Research Strategy

Methods

  • Context observation of family story-sharing around keepsakes
  • Semi-structured interviews on memory loss and communication gaps
  • 24-participant mixed-role questionnaire (Grandchild subgroup ages 18-22) and prototype walkthrough
  • Behavioral metrics: task success, completion time, recommendation score

Literature Alignment

We aligned our protocol with prior work on tangible memory sharing, family memory curation, and intergenerational interaction design [1][2][3]. We also considered reflective memory interfaces and slow technology principles for sustained engagement [4][5].

User Journey Map

Current pain points and how TimeLens addresses each stage

1

Discover

Action: Finds a meaningful keepsake at home

Pain: Object has no context; story is locked in memory

Solution: NFC tag links physical object to digital memory

2

Tap / Scan

Action: Taps phone to NFC tag or uses manual fallback

Pain: Older users unsure where to tap; devices vary

Solution: Large tap target + visual feedback + fallback path

3

Reveal

Action: Watches 3D memory scene unfold

Pain: Text archives feel dry and unengaging

Solution: Playful 3D reveal with story card and emotional audio

4

Reply

Action: Reads and sends cross-generation messages

Pain: One-way storytelling dies after a single telling

Solution: Lightweight reply loop keeps memory alive

5

Revisit

Action: Returns to memory map over time

Pain: Memories are scattered and hard to find again

Solution: Collection map with unlock progress and revisit shortcut

Journey insight:

The strongest design signal is that interaction must start from the physical object, not a menu. This aligns with tangible interaction research [1][3].
