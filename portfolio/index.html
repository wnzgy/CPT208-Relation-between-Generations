Project Overview

CPT208 - Relation Between Generations

🎯 Problem

Family stories stay hidden inside everyday objects. Children can see the keepsake, but not the memory behind it. Many existing tools feel archival and text-heavy.

💡 Solution

TimeLens turns a physical object into a playful story portal. Tap an NFC-tagged family keepsake to unlock a visual memory reveal and leave a lightweight reply across generations.

👥 Target Users

  • Grandparents (60+) who want to share family stories
  • Grandchildren (10-15) curious about family history
  • Parents bridging the generational gap

🔧 Technologies

  • Web NFC API for tangible interaction
  • React Three Fiber for 3D visualization
  • Responsive mobile-first design
  • LocalStorage for data persistence
