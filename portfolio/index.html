Project Overview

CPT208 - Relation Between Generations | Group B4-1

Problem

Family stories are often attached to physical keepsakes, but those stories are rarely preserved in an engaging way for younger generations.

Solution

TimeLens combines NFC-tagged objects, playful 3D memory reveal, and lightweight messaging to support two-way storytelling across generations.

Target Users

  • Grandparents who want to pass down stories
  • Grandchildren who prefer interactive learning
  • Parents as communication bridge builders

Deliverables

Questionnaire + 50 samples Data statistics dashboard Design process portfolio Web prototype entry
